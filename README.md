### Project - Logarithmic Regression & Random Forest - LAPD Violence

<h4><b><u>This project explores the LAPD NIBR dataset.</b></u><br></h4>
<p>The National Incident-Based Reporting System (NIBRS) is the new FBI crime reporting system implemented in 2021.<br>
It took time for forces to implement, with the LAPD only reporting in this manner as of 2024.<br><br>
It moves away from the previous Summary Reporting System (SRS) in one key way - actual crimes are reported - not the summary statistics.<br><br>
The FBI say this system 'can provide more useful statistics to promote constructive discussion, measured planning, and informed policing'.<br>[Link Here](https://www.fbi.gov/how-we-can-help-you/more-fbi-services-and-information/ucr/nibrs?__cf_chl_f_tk=LZIAjagxbUe4BEHK3cSyfQBzG3o2a51uhmez4MY4b30-1783271458-1.0.1.1-Lev7EIpou.m14UpP9IUc7AUAR8AmjFa5O459hNNnbLA, "FBI NIBRS Summary")</p>
