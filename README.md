#### **This project explores the LAPD NIBR dataset.**

The National Incident-Based Reporting System (NIBRS) is the new FBI crime reporting system implemented in 2021. It took time for forces to implement, with the LAPD only reporting in this manner as of 2024.

It moves away from the previous Summary Reporting System (SRS) in one key way—actual crimes are reported, not the summary statistics.

The FBI say this system "can provide more useful statistics to promote constructive discussion, measured planning, and informed policing".
